# CAG MAP

## About

The goal of the CAG MAP is to provide a guide for those on a computer *aided* governance journey.

## Development

To deploy the [MKdocs](https://www.mkdocs.org/) locally:

- `pip install mkdocs`
- `git clone https://github.com/BlockScience/cag-map.git`
- `cd cag-map`
- `mkdocs serve`
